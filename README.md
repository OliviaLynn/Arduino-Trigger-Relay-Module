## Installation

- Simply clone this repo to your local machine using `https://github.com/OliviaLynn/Arduino-Trigger-Relay-Module.git`

### Running
- Run from your shell:
```shell
$ bash <path/to/RefreshingSendPulseToRelayModule.sh>
```
